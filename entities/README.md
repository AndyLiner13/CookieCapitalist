# Entity Configuration Files

This folder contains entity configuration instruction files generated by the `configure_entity` MCP tool.

Each file defines how an entity should be configured in the Horizon Worlds Desktop Editor.

## Usage

Use the `configure_entity` tool to create new entity configurations:

```typescript
// List available entity types
configure_entity("", "")

// Get schema for an entity type
configure_entity("", "Empty_Object")

// Create a configuration
configure_entity("MyEntity", JSON.stringify({
  "entityType": "Empty_Object",
  "properties": {
    // ... properties
  }
}))
```

The tool will validate your configuration and create an instruction file here.
